This application is a Python-based multi-threaded command-line port scanner designed to efficiently identify open and closed TCP ports on a target host. By leveraging Python’s threading capabilities, the tool can scan multiple ports simultaneously, significantly improving scanning speed and performance compared to traditional single-threaded scanners.

The scanner attempts to establish TCP connections to a range of specified ports and reports their status, helping users quickly identify accessible services running on a system. Its lightweight command-line interface makes it easy to integrate into security testing workflows and network diagnostics.

This project demonstrates practical knowledge of network programming, socket communication, concurrency using threads, and basic cybersecurity reconnaissance techniques.
